# NStreamIng
Ce site de Streaming a été réalisé dans le but d'un mini projet de NSI, il n'est en aucun cas destiné au visionnage de contenu dont vous n'avez pas les droits.

## Utilisation: 
Lors du lancement de l'exécutable, vous devrez vous rendre à l'adresse https://0.0.0.0:5000 ou à une des adresses donnée par le programme, le site sera accessible seulement depuis votre réseau et depuis tout les autres appareils connectés, cette adresse est accessible
